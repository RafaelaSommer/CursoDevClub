# 🚀 Gerador de Página com IA

Um gerador de páginas web utilizando Inteligência Artificial para criar layouts modernos, responsivos e personalizados a partir de descrições em linguagem natural.

## ✨ Funcionalidades

- Geração automática de páginas HTML.
- Criação de layouts responsivos.
- Personalização de cores, fontes e componentes.
- Integração com APIs de IA.
- Exportação dos arquivos gerados.
- Interface intuitiva e fácil de usar.

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Node.js
- API de Inteligência Artificial
- Bootstrap/Tailwind CSS (opcional)

## 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/RafaelaSommer/CursoDevClub.git
```

Acesse a pasta do projeto:

```bash
cd CursoDevClub
```

Instale as dependências:

```bash
npm install
```

## ▶️ Executando o Projeto

Inicie o servidor:

```bash
npm start
```

Ou em modo desenvolvimento:

```bash
npm run dev
```

Acesse:

```text
http://localhost:3000
```

## 💡 Como Utilizar

1. Digite uma descrição da página desejada.
2. Clique em **Gerar Página**.
3. A IA criará automaticamente a estrutura da página.
4. Edite e personalize conforme necessário.
5. Exporte os arquivos gerados.

### Exemplo de Prompt

```text
Crie uma landing page para uma clínica odontológica com:
- Cabeçalho moderno
- Seção de serviços
- Depoimentos de clientes
- Formulário de contato
- Rodapé com redes sociais
```
